WSS

The WSS class is used internally only. It holds information about a WebSocket Stream, which clients can use for real-time updates. For more info, see WebSockets.

Properties

Property Type Description
_id string Unique identifier for the WebSocket session.
Socket WebSocket WebSocket connection object.
CurrentQueue string ID of the current queue being managed.
DeviceName string Name of the device connected via WebSocket.
UserId string ID of the user associated with the session.
IsPublic bool Indicates if the WebSocket stream is public.
SendProgress bool Indicates if progress updates should be sent.
LastPing DateTime Timestamp of the last ping from the client.
